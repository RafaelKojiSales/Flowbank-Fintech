# FlowBank - Projeto Fintech

FlowBank é uma interface web responsiva desenvolvida como parte de uma atividade acadêmica de Fintech. O projeto apresenta uma tela de dashboard financeiro com foco em organização, clareza visual e acompanhamento das principais informações do usuário.

A tela permite visualizar saldo disponível, receitas, despesas, economia mensal, limite do cartão, fluxo mensal, categorias de maiores gastos, meta de reserva de emergência e últimas movimentações financeiras.

O objetivo do projeto é demonstrar a construção de uma interface moderna, funcional e responsiva utilizando HTML, CSS e Tailwind CSS, mantendo os arquivos organizados e separados conforme as boas práticas de desenvolvimento front-end.

## Tecnologias utilizadas

* HTML5
* CSS3
* Tailwind CSS
* Design responsivo

## Funcionalidades representadas

* Visualização de saldo disponível
* Indicadores financeiros
* Acompanhamento de receitas e despesas
* Categorias de gastos
* Meta de reserva de emergência
* Extrato recente
* Interface adaptável para dispositivos móveis
